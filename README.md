# paradise-ticket-web
사내 구내식당 식권 사용 및 월별, 사용자별 데이터 제공을 위한 서비스입니다.

## 개발 환경
- Java 1.8
- Gradle
- Thymeleaf
- jQuery
- HTML
- CSS
